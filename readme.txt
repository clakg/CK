- ls (list)
- cd/dir (change directory)

Initialiser un paquet NodeJS :
- npm init

Installer un paquet (dependencie) :
- npm install lenomdupaquet

Installer un paquet de développement (Devdependencies) :
- npm install lenomdupaquet --save-dev

Désinstaller un paquet :
- npm remove lenomdupaquet

Lister les paquets :
- npm list

Lancer un script :
- npm run nomdelatache
